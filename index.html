Описание проекта

Проект под названием Media Costs — это сложное веб-приложение для планирования рекламных кампаний. Приложение загружает необходимые данные с сервера и предоставляет пользователю инструменты для работы с ними. Ключевой функциональностью является таблица, аналогичная Excel, которая позволяет создавать рекламные блоки (флайты) с настраиваемыми свойствами, производить расчеты, а также задавать внешний вид флайтов для их визуализации.

Обязанности и ключевые задачи

  • Разработка новой функциональности, оптимизация, исправление ошибок и рефакторинг кода;
  • Участие в командных встречах и обсуждение деталей разработки;
  • Решение проблемы с зависанием интерфейса, что привело к значительному повышению производительности приложения;
  • Написание юнит и интеграционных тестов;
  • Разработка универсальной логики для работы с формами на основе Redux Form с поддержкой каскадных обновлений;
  • Разработка интерактивного элемента Formula Constructor - компонент учитывая контекс формулы предсказывает следующий шаг и упрощает механизм написание формул для пользователя;
  • Создание системных типов данных для работы с формулами на уровне приложения;
  • Реализация механизма зависимостей для оптимизации перерасчета формул;
  • Добавление глобальных горячих клавиш и менеджера навигации.
